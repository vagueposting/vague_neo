home

welcome!

hi, viewer, and welcome to summer snow!

first things first, i go by many names. check the ABOUT link above to learn more about the webmaster.

i use this site to archive to and link to my work. and also so i can build shrines to the stuff i love. i have a lot of creative material and it would be a shame to put them in all in the extremely transient land of social media.

sorry if this place looks downright hideous. as you can see, i’m no good with html. promise you that i’m just frankensteining the code i see to make something relatively servicable. i would LOVE to say that i made this whole site from scratch but sadly that's not the case haha in december of 2024, i picked up the odin project’s programming classes, which pushed me to revitalize this dinky website.

what you’ll find here

like i said earlier, i use this website to archive my works and collate things which exist on otherwise disparate websites. i also use this to host small web apps that i am particularly proud of.

i like to think that the website on its own is pg-13, as are most of the works that will be published. r-15 works will come with a warning.

i develop games on twine sugarcube as a hobby. once i’ve learned what i need to learn on javascript i’ll probably shift slightly towards python so that i can figure out ren'py.

i don't exactly have a set theme for this website. if you're here by 2026 and there’s junk like air fryer chicken recipes, updates on my projects, or articles about arknights, then i am so sorry. (i am not sorry.)
